# Monyvi Mobile App

React Native mobile app built with Expo.

**Note:** All dependencies are managed in the root `package.json`.

## Running the App

From project root:

```bash
# Start Expo dev server
npm run mobile

# Run on iOS
npm run mobile:ios

# Run on Android
npm run mobile:android
```

## Tech Stack

- **Framework**: Expo ~52.0
- **React Native**: 0.76.5
- **Navigation**: Expo Router
- **Styling**: NativeWind (Tailwind)
- **Database**: WatermelonDB
- **Voice**: @react-native-voice/voice

## Features

- 📊 Dashboard with Egyptian theme
- 🎤 Voice transaction input
- 📱 Notification parsing
- 💰 Multi-pocket management
- 🏆 Multi-currency (EGP/USD/Gold)

## Installation

Run from project root:

```bash
npm install
```

## Development

1. Install Expo Go on your phone
2. Run `npm run mobile`
3. Scan QR code with Expo Go
